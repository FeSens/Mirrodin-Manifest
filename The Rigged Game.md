# The Rigged Game

## Card Type Line
Sorcery

## Rules Text
> Scry 2, then gamble.
>
> If you win the gamble, compound twice.
>
> If you lose the gamble, redistribute.

## Flavor Text
> *Win, and the wealth compounds. Lose, and everyone shares the pain. Either way, you're never the one who suffers.*
